# SoftwareInc.Deutsch
Deutsche Übersetzung für Software Inc. Alpha 8 (8.2)



Installation: 
Das Verzeichnis "Deutsch" in das Spielverzeichnis "Software Inc\Localization" verschieben

Im Spiel kann dann Deutsch oben links ausgewählt werden.
